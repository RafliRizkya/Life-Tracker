# AI Tools

This project has multiple AI tools available.

Choose the correct tool for the task.

---

## Context7

Purpose

Official documentation.

Use when:

- React
- Next.js
- Tailwind
- Zustand
- Framer Motion
- Recharts
- FastAPI
- Supabase
- external npm packages

Never rely on memory if Context7 is available.

---

## Ponytail

Purpose

Engineering quality.

Apply during every implementation.

Focus on:

- simplicity
- readability
- maintainability
- small diffs

Avoid:

- over engineering
- unnecessary abstractions

---

## Motion

Purpose

Planning.

Use Motion when:

- large feature
- roadmap
- backlog
- implementation planning

Do not use Motion for tiny fixes.

---

## Playwright

Purpose

Validation.

Use after frontend changes.

Verify:

- UI
- navigation
- responsiveness
- console errors
- regressions

---

## Frontend Design Skill

Purpose

Improve UI quality.

Use when:

- creating pages
- improving layouts
- improving spacing
- improving typography
- improving UX

---

## Webapp Testing

Use when user flows change.

Validate:

- loading
- empty
- error
- success

Test realistic scenarios.

---

## General Rule

Do not use tools because they exist.

Use them because they improve the outcome.
