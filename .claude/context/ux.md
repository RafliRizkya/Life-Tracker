# UX Principles

The user should think about life.

Not about the interface.

---

# Reduce Decisions

Every extra decision increases friction.

Reduce unnecessary choices.

---

# One Primary Action

Every screen should have one obvious primary action.

Avoid multiple competing CTAs.

---

# Progressive Disclosure

Hide complexity.

Reveal advanced options only when needed.

---

# Smart Defaults

Default values should be correct most of the time.

Reduce typing.

Reduce clicks.

---

# Consistency

If one interaction behaves a certain way,

similar interactions should behave the same.

---

# Feedback

Every user action deserves feedback.

Examples

Saving...

Saved.

Deleted.

Updated.

Import completed.

---

# Undo

Destructive actions should support recovery whenever possible.

---

# Forms

Forms should feel conversational.

Avoid intimidating layouts.

Group related information.

---

# Validation

Validate early.

Explain clearly.

Never blame the user.

---

# Empty Screens

An empty screen is an opportunity.

Teach.

Guide.

Encourage.

---

# Delight

Small moments matter.

Subtle animations.

Thoughtful wording.

Meaningful illustrations.

Avoid gimmicks.

---

# Trust

Users should always know:

- where they are
- what changed
- what will happen next

Uncertainty destroys trust.
