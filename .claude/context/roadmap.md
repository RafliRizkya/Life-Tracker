# Future Vision

Become a true Personal Operating System.

The application should understand:

- goals
- finances
- habits
- learning
- career
- reflections

and connect them intelligently.

---

# Decision Making

Every new feature should strengthen existing workflows.

Avoid isolated features.

Prefer systems over tools.

---

# Feature Evaluation

Ask:

Does this improve decision making?

Does it reduce cognitive load?

Will users still appreciate it after one year?

If not,

reconsider.
