# Performance Checklist

## React

- [ ] Avoid unnecessary state.
- [ ] Avoid prop drilling.
- [ ] Expensive computations memoized when needed.

---

## Network

- [ ] Payload minimized.
- [ ] Lazy loading.
- [ ] Caching considered.

---

## Rendering

- [ ] No layout shift.
- [ ] Fast initial render.
- [ ] Skeletons where appropriate.

---

## Bundle

- [ ] No unnecessary dependency.
- [ ] Bundle impact reviewed.
