# Database Checklist

## Schema

- [ ] Naming consistent.
- [ ] Relationships explicit.
- [ ] No duplicated fields.

---

## Migration

- [ ] Existing data preserved.
- [ ] Rollback possible.
- [ ] Migration tested.

---

## Integrity

- [ ] Constraints correct.
- [ ] Validation enforced.
- [ ] IDs unique.

---

## Security

- [ ] RLS enabled.
- [ ] Owner-only access.
- [ ] No unnecessary permissions.

---

## Performance

- [ ] Indexes considered.
- [ ] Query cost reviewed.
