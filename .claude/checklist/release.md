# Release Checklist

## Product

- [ ] Feature complete.
- [ ] UX polished.
- [ ] Copy reviewed.

---

## Code

- [ ] Lint.
- [ ] Build.
- [ ] Type check.
- [ ] No dead code.

---

## Testing

- [ ] Playwright.
- [ ] Manual testing.
- [ ] Responsive.

---

## Accessibility

- [ ] Keyboard.
- [ ] Contrast.
- [ ] Focus.

---

## Performance

- [ ] Bundle acceptable.
- [ ] Loading smooth.

---

## Security

- [ ] Secrets protected.
- [ ] Validation reviewed.

---

## Documentation

- [ ] README updated.
- [ ] Roadmap updated.
- [ ] Changelog updated if needed.

---

## Git

- [ ] Generate commit message.
- [ ] Never commit automatically.

---

## Final Question

Would I confidently ship this to production today?

If the answer is "no",

identify why before releasing.
